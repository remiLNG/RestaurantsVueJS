# RestaurantsVueJS

# Prerequisites

+ Node Js
+ Vue Js
+ MongoDB

# Run

+ install all the dependencies
```
npm i 
```
+ Move to server repository

```
cd serveur
```

+ Start the server
```
node serverCrudWithMongo.js
```


+ Move to client repository
```
cd client
```
```
npm i -g http-server or npm i -g serve
```
+ Start the client 
```
http-server or serve
```
